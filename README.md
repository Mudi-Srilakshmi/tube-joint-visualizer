Tube Joint Visualizer:

A simple Electron-based desktop application that visualizes tube joints with interactive controls.
Users can view the tube joint structure, rotate it, and explore the visualization in an easy graphical interface.

Features:

Visualize tube joints in 3D
Rotate and inspect the joint from different angles
Simple and clean user interface
Built using Electron

Project Structure:
 
main.js          -> Electron main process
app.js           -> Application logic
index.html       -> UI layout
src/             -> Additional scripts and assets
package.json     -> Project configuration

How to Run:

Clone the repository:
git clone https://github.com/Mudi-Srilakshmi/tube-joint-visualizer

Go into the project:
cd tube-joint-visualizer

Install dependencies:
npm install

Run the application:
npm start

Build:
 
npm run build:win
Building an installer may require additional Windows signing configuration.

Notes:

Running the app with npm start works without any installer.
The project is created only for demonstration and learning purposes.
